## zsh setup

### Requirements

- zsh
- [Homebrew](https://brew.sh/)
- [Antigen](https://github.com/zsh-users/antigen) - `~/.local/bin/antigen.zsh` に配置
- [starship](https://starship.rs/)
- [Neovim](https://neovim.io/) - vim alias用
- [pyenv](https://github.com/pyenv/pyenv)
- [ghcup](https://www.haskell.org/ghcup/) - Haskell toolchain
- [mise](https://mise.jdx.dev/) - runtime version manager
- [fzf](https://github.com/junegunn/fzf) - `~/.local/bin/.fzf.zsh` に配置
- [ripgrep](https://github.com/BurntSushi/ripgrep) - fzfのデフォルトコマンド用
- [lazygit](https://github.com/jesseduffield/lazygit)
- libpq - `brew install libpq`
- [Google Cloud SDK](https://cloud.google.com/sdk) (optional)

## Neovim setup

### Requirements

- [Neovim](https://neovim.io/) >= 0.9.0
- Git
- [Nerd Font](https://www.nerdfonts.com/) - アイコン表示用
- C compiler (gcc/clang) - treesitter用
- [Node.js](https://nodejs.org/) + yarn - markdown-preview.nvim用
- [ripgrep](https://github.com/BurntSushi/ripgrep) - telescope用
- [fd](https://github.com/sharkdp/fd) - telescope用 (optional)
- [lazygit](https://github.com/jesseduffield/lazygit) - lazygit.nvim用
- [gh CLI](https://cli.github.com/) - gh.nvim用
- [GitHub Copilot](https://github.com/features/copilot) subscription (optional)
